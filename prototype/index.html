This is a paragraph element, it is a block element providing padding and setting itself on a new line from other content. This is an emphasis element. it italicizes words usually within another text based element. Unlike the paragraph "p" element the "em" element is inline meaning it stays with the flow of surrounding content. It does not take upo full horizontal space nor does it place itself on a new line. This is an element called strong it works just like the emphasis element except it is for bolding a word or phrase rather than italicizing. It has the highest emphasis of the two.
